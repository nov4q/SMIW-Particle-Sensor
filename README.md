# University project containing schemas and code for Home particle sensor.

- schemas were created using Eagle software;

Parts:
- PLantower PMS7003;
- ESP32-DevKitC 32E

Features:
- Measuring PM2.5 and PM10 particles;
- Wi-Fi connection;
- data upload to Thingspeak.
